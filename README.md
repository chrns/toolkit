# Toolkit

> *One toolkit to calculate them all.*

Toolkit gathers essential calculators for Embedded System Engineers in one convenient place. Whether you need to quickly compute Ohmâ€™s law, impedance, battery life, or more, Toolkit has you covered.

This project is PWA-ready and can be installed on your mobile device for quick access anytime.

## Installation

- **Android (Chrome/Edge):** Open [Toolkit](https://toolkit.chrns.com/) in your browser, then tap the menu and select **Add to Home screen**.
- **iOS (Safari):** Open [Toolkit](https://toolkit.chrns.com/) in Safari, tap the **Share** button, then select **Add to Home Screen**.

Once installed, Toolkit works offline, so you can use it without an internet connection.
